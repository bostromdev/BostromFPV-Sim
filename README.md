# BostromFPV-Sim
This FPV Drone Sim is in v1.0 and built around the feeling of flying an actual quad—not an arcade drone game. It focuses on responsive freestyle control, true quadcopter aerodynamics, Betaflight-style PID behavior, propwash, momentum, motor response, and customizable controller mapping for radios like the TX16S. Great for older computers/macbooks.
