# Bostrom FPV Simulator

A free FPV simulator built for responsive freestyle practice, technical lines, custom environments, and a direct FPV control feel.

## Current Release

**Bostrom v1.0**

## Features

- FPV-style flight controls for freestyle practice
- Freestyle-focused environments and technical lines
- Adjustable rates, PID-style tuning, camera, and HUD settings
- Configurable HUD and crosshair options
- Ongoing development with room for new maps, aircraft profiles, and improvements

## Getting Started

1. Download the latest release for your platform.
2. Open the application.
3. Configure your controller and settings in-game.
4. Fly.

## Controls

Controls can be configured in the in-game settings menu.

## Status

Bostrom FPV Simulator is actively being developed. The current focus includes improving map quality, fixing visual glitches, and cleaning up collision issues so environments feel smoother and more reliable to fly.

## Feedback and Bug Reports

Feedback is appreciated, especially reports of map glitches or collision problems. If you find something off, please include the map area, what happened, and a screenshot or short clip when possible. That makes it much easier to reproduce and fix.
